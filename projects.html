Detection of Aimbotting in FPS games using ML

This dissertation explores the detection of cheating in first-person shooter (FPS) games through machine learning techniques, focusing on aim-bot detection and mouse dynamics. Various models, including MLPClassifier, Logistic Regression, and Decision Tree Classifier, were evaluated to assess their effectiveness in identifying cheating behavior.

The results demonstrate that the MLPClassifier performs well, achieving accuracy levels around 80%, which aligns with the study’s goals. While promising, the research highlights the need for further improvements and ethical considerations in data collection methods. Overall, the project aims to enhance anti-cheat systems within FPS games using advanced machine learning approaches.
